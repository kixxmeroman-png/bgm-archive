유튜브 재생목록 일괄 등록 (웹페이지 소스 방식)

1. 등록을 원하는 유튜브 재생목록 페이지를 엽니다.

2. 키보드에서 Ctrl + U를 누르면 새로운 탭에 복잡한 소스 코드 화면이 나타납니다.

3. 그 화면에서 Ctrl + A (전체 선택) 한 뒤, Ctrl + C (복사) 합니다.

4. 복사한 내용 전체를 아래 입력란에 Ctrl + V (붙여넣기) 하시면 안전하게 모든 영상을 파싱해옵니다.
